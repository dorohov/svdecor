Почему светильники из гипса пользуются спросом?

Для этого есть несколько причин:

  1. 1. Гипоаллергенность и экологичность. Их можно использовать в детских комнатах, больницах.
  2. 2. Универсальность. Из гипса можно сделать светильник любой формы, выкрасить его под цвет интерьера.
  3. 3. Негорючесть. Гипс не воспламеняется (класс НГ (КМ-0)), нет ограничений на монтаж в коммерческих помещениях.
